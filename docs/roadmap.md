## 1.0（正式版）以降

以下は beta 卒業後に検討する。beta での混乱を避けるため着手しない。

- PDF / PNG エクスポート・印刷

- 自動アップデータ（Tauri Updater）
- テンプレートのエクスポート・インポート（研究室内共有）
- 高度な共同編集・差分比較
- ダークモード
- 日英多言語

- `.ptcl` schemaVersion 1.0 が stable（破壊的変更なし）
- README・使い方ドキュメントが整備済み
