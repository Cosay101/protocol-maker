ゴシック体フォントファイルをここに配置してください。

対応フォーマット: .woff2（推奨）/ .ttf / .otf

ファイル名の規則:
  regular.woff2  (または regular.ttf / regular.otf)  ← 標準ウェイト
  bold.woff2     (または bold.ttf    / bold.otf   )  ← 太字ウェイト

例: NotoSansJP-Regular.woff2 → regular.woff2 にリネームして配置

@font-face の定義は app/globals.css に記述済みです。
ファイルを配置すると即座に反映されます。
